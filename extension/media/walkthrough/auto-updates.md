## Stay Up to Date

When someone pushes changes to the CursorPlugins repo and you pull them, the extension can detect the update and prompt you to re-sync.

### Options

- **Check on startup** — Compares your deployed commit hash against the repo's current HEAD each time you open a workspace.
- **File watcher** — Monitors the repo's git refs so you're notified immediately after a `git pull`.

Enable startup checks in settings: `cursorTeamTools.checkForUpdatesOnStartup`
