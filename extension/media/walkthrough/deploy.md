## Deploy Configuration

Click **Sync Now** to deploy the `.cursor/` directory to your workspace. The extension will:

1. Detect whether you're in single-repo or multi-repo mode
2. Resolve which profiles apply based on your repo mappings
3. Copy the matching skills and rules into `.cursor/`
4. Set rule scoping automatically (alwaysApply or globs)

You can re-run this any time from the command palette: **Cursor Team Tools: Sync Config**
