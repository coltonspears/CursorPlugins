# Team Plugin

Shared Cursor skills and rules for the team's WPF and SQL Server projects.

## Profiles

Assets are tagged with `profiles` in their YAML frontmatter:

| Profile | Description | Repos |
|---------|-------------|-------|
| *(common)* | Applied to all repos | All |
| `wpf` | WPF/XAML MVVM patterns and conventions | WPFApp, WPFAdmin |
| `sql` | SQL Server naming and review conventions | SqlServerDB |

Files without a `profiles` tag are treated as common.

## Skills

- **code-review** — Structured code review workflow (common)
- **wpf-patterns** — MVVM patterns and data binding guidance (wpf)
- **sql-review** — SQL script and migration review (sql)

## Rules

- **coding-standards.mdc** — Team-wide coding standards (common)
- **wpf-conventions.mdc** — WPF/XAML conventions (wpf)
- **sql-conventions.mdc** — SQL Server conventions (sql)
